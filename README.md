# OMNITRIX – Ben 10 Themed Interactive Website

An immersive Ben 10 themed website featuring a fully interactive 3D Omnitrix, alien playlist, Omnitrix variants, and character profiles.

## Features
- Interactive 3D Omnitrix (activation animations + sound)
- Classic alien playlist (original 10 aliens)
- Omnitrix variants with 3D models
- Core character showcase
- Smooth animations + particle effects

## Tech Stack
- React + TypeScript
- Vite
- Three.js

## Live Demo
https://poseidonsben10site.netlify.app/

## Setup
npm install  
npm run dev

npm run build
